Install
-------
Latest released version (`PyPI package <http://pypi.python.org/pypi/sapyens>`_):

.. code-block::  console

    pip install sapyens

Development version:

.. code-block::  console

    pip install git+https://github.com/fillest/sapyens.git#egg=sapyens
